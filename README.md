
# myshows_api_tests


## <font color="green"> tests </font>
### <font color="blue"> test_api_series.py </font>
В директории tests и в файле test_api_series.py лежат тесты.

## <font color="green"> config/api_config.py </font>

В этой директории лежат различные конфигурационные параметры - стенды, урлы, токены и т.д.

## <font color="blue"> conftest.py </font>

файл conftest нужен для того, чтобы в нем pytest первоначально искал различные вспомогательные сущности, например, фикстуры (т.е. ф этом файле прописан путь к сущностям. По этому пути Python ищет нужные файлы).

## <font color="green"> schemas </font>

В директории schemas лежат json-файлы, содержащие схемы ответов в специальном формате.

## <font color="green"> ApiSession </font>

В директории ApiSession лежит файл api_session.py.

## <font color="blue"> .env </font>

В файле .env лежит подключение к БД и бэку.
 
 ## <font color="green"> data/series.sql </font>

 В директории data лежит скрипт для БД.
 
 ## <font color="blue"> utils.py </font>

 Файл utils.py - это файл, где лежат различные утилиты.
